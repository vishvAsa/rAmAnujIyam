सन्निधि-शिष्या उच्यन्ते - नृसिंह-मूलक-सन्न्यास-परम्परायाः।  

किन्तु तेषां भाष्योपदेश-परम्परा, रहस्योपदेश-परम्परा च घटिकाश्रुति-शतकाद् उद्गच्छति।

<details><summary>आदिमाहोबिलमठाचार्यस्य काले रविलोचनः</summary>

अन्यत्र द्रष्टव्यम्। 
</details>

 भूतशुद्धिमन्त्रेषु व्यत्यासो विद्यते -  
तत्राहोबिलपक्षो वरो भाति।

## समाश्रयण--भगवद्-विषय-कालक्षेप-परम्परा
रामानुजः → ऎम्बार् गोविन्दः → पराशर-भट्टः → पॆरिय-जीयर्, नञ्-जीयर् (मैसूरु-प्रदेशे तॆङ्गले-ग्रामात्) 9k-व्याख्या-स्रोतः (शिष्य-लेखितः) → नम्पिळ्ळै → पॆरियव्-आच्चाऩ्-पिळ्ळै (दिव्य-प्रबन्ध-व्याख्याता), वडक्कुत्-तिरु-वीदिप्-पिळ्ळै (ईडु-३६k-व्याख्या-लेखकः) → (भगवद्-विषय-कालक्षेपे हि +++(दत्त्वा=)+++ईय्-उन्नि-माधवः →) श्रीरङ्गः → केशवः → श्रीनिवासः → केशवः → प्रथमशठकोपः।

एवं प्रथमाहोबिलाचार्यात् पूर्वं तत्-पूर्वज-द्वारा लोकाचार्य-पितरं वडक्कु-तिरुवीदि-पिळ्ळै-महाशयम् प्राप्नोति।
